B) Créer le StackedWidget :
   1. Supprimer tous les widgets du centralwidget
   2. Ajouter un QStackedWidget nommé "stackedWidgetMain"
   3. Taille: 1298 x 730 pixels
   4. Remplir toute la fenêtre

C) COPIER VOTRE PAGE LOGIN :
   1. Ouvrir VOTRE fichier login.ui original (autre Qt Designer)
   2. Sélectionner TOUT (Ctrl+A)
   3. Copier (Ctrl+C)
   4. Retour dans smartmarket.ui
   5. Cliquer sur "Page 0" du StackedWidget
   6. Coller (Ctrl+V)
   
   ✅ VOTRE LOGIN EST LÀ AVEC 100% DU STYLE !

D) COPIER VOTRE PAGE PUBLICATION :
   1. Clic droit sur StackedWidget → "Insérer une page"
   2. Ouvrir VOTRE fichier publication.ui original
   3. Sélectionner TOUT (Ctrl+A)
   4. Copier (Ctrl+C)
   5. Retour dans smartmarket.ui
   6. Cliquer sur "Page 1" du StackedWidget
   7. Coller (Ctrl+V)
   
   ✅ VOTRE PUBLICATION EST LÀ AVEC 100% DU STYLE !
